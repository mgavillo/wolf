# wolf
Wolf is a 42 project.

run it with `make` and `./wolf3d ./map/map1`. You can do your own map as well.

 _|^|

|<|v|>| move front, back, left and right

|P| come back to the menu
|A| down
|Z| up

:mouse: use the mouse to look around.
